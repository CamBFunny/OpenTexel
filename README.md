# OpenTexel
Open Source recreation of Defender of Texel

I will be using the [official Wiki](https://defenderoftexel.fandom.com/wiki/Defenderoftexel_Wiki) to research.

## Prototype
Most popular Texel characters found here: [https://defenderoftexel.fandom.com/wiki/AB13_Fighter_Rankings](https://defenderoftexel.fandom.com/wiki/AB13_Fighter_Rankings) \
Use alphabet monsters to randomly generate a team of 9 fighters. \
